# 运行和部署您的AI Studio应用

这里包含了您在本地运行应用所需的一切。

在AI Studio中查看您的应用：https://ai.studio/apps/temp/1

## 本地运行

**先决条件：** Node.js


1. 安装依赖：
   `npm install`
2. 在 [.env.local](.env.local) 文件中设置 `OPENROUTER_API_KEY` 为您的OpenRouter API密钥
3. 运行应用：
   `npm run dev`
